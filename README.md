# Growtopia Pixel Art Generator

Turn any image into a buildable Growtopia pixel art design — matches every pixel to the closest-colored in-game item and gives you a build grid, preview images, shelf layouts, and a shopping list.

[English](#english) | [中文](#中文)

## English

### What's in here

| File | What it does |
|---|---|
| `SolidItemFilter.py` | Filters the full Growtopia item list down to "solid" items usable for pixel art (checks each sprite's fill %) → `output/solid_items.json` |
| `ComputeItemStats.py` | Pre-computes the average color + variance for every solid item's sprite → `output/item_stats.json` |
| `PixelMatcher.py` | matches an image's pixels to items and generates the build grid, previews, and Excel sheet |
| `gtCommon.py` | Shared color-math helpers used by the two scripts above |

### Setup

1. Clone the repo.
2. Create a `sprites/` folder in the project root and drop in your Growtopia item sprite `.png` files (not included in this repo — see note below).
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

### Usage

Run these in order:

```bash
# 1. Build the list of usable solid items (only needs to be re-run if your sprite set changes)
python SolidItemFilter.py

# 2. Cache each item's average color (only needs to be re-run if sprites change)
python ComputeItemStats.py

# 3. Generate pixel art from an image
python PixelMatcher.py your_image.png 70 70
```

Output lands in `output/pixelmatcher/<image_name>/` and includes:
- `*_grid.json` / `*_grid_old.json` — the build grid (two matching algorithms)
- `*_preview.png` / `*_preview_old.png` — flat preview of the build
- `*_comparison.png` — 6-panel side-by-side comparison
- `*_shelves.png`, `*_shelves_charcoal.png`, `*_shelves_old.png` — how it looks on in-game display shelves
- `*_build.xlsx` — build grid + shopping list in Excel

Optionally, add `output/blacklist.json` to exclude specific items or name patterns from matching:
```json
{
  "banned_patterns": ["gem", "vip"],
  "banned_items": ["Rare Item Name"]
}
```

### The `output/` folder

| File | What it is |
|---|---|
| `blacklist.json` | Hand-edited list of items/patterns to exclude from matching |
| `solid_items.json` | Generated by `SolidItemFilter.py` — don't edit by hand |
| `item_stats.json` | Generated by `ComputeItemStats.py` — don't edit by hand |
| `PLEASE READ.txt` | Explains why the sprites aren't in this repo |

### A note on sprites

The `sprites/` folder isn't included in this repo. Growtopia's item sprites are the game's own copyrighted assets, not something this project has the right to redistribute — you'll need to supply your own copy locally. See `output/sprites_note.txt` for details.

## 中文

### 这个仓库里有什么

| 文件 | 作用 |
|---|---|
| `SolidItemFilter.py` | 从Growtopia全部道具里,筛出能拼像素画的"实心"道具 → 输出 `output/solid_items.json` |
| `ComputeItemStats.py` | 提前把每个实心道具贴图的平均颜色和方差都算好,存起来 → 输出 `output/item_stats.json` |
| `PixelMatcher.py` | 主程序,把图片的每个像素匹配成对应道具,生成建造网格、预览图和Excel表 |
| `gtCommon.py` | 上面两个脚本共用的颜色计算工具函数 |

### 环境搭建

1. 把仓库clone下来。
2. 在项目根目录建一个 `sprites/` 文件夹,把你自己的Growtopia道具贴图(`.png`)放进去(仓库里不包含这些贴图,原因见下面说明)。
3. 装依赖:
   ```
   pip install -r requirements.txt
   ```

### 使用方法

按顺序跑这几个脚本:

```bash
# 1. 生成可用的实心道具列表(贴图没变的话不用重复跑)
python SolidItemFilter.py

# 2. 缓存每个道具的平均颜色(贴图没变的话不用重复跑)
python ComputeItemStats.py

# 3. 拿一张图生成像素画
python PixelMatcher.py your_image.png 70 70
```

生成的东西都在 `output/pixelmatcher/<图片名>/` 里,包括:
- `*_grid.json` / `*_grid_old.json` —— 建造网格(两种匹配算法各一份)
- `*_preview.png` / `*_preview_old.png` —— 平铺预览图
- `*_comparison.png` —— 六宫格对比图
- `*_shelves.png`、`*_shelves_charcoal.png`、`*_shelves_old.png` —— 放在游戏里货架上大概长什么样
- `*_build.xlsx` —— 建造网格 + 购物清单的Excel表

如果想屏蔽掉某些道具,可以加一个 `output/blacklist.json`:
```json
{
  "banned_patterns": ["gem", "vip"],
  "banned_items": ["Rare Item Name"]
}
```

### `output/` 文件夹里有什么

| 文件 | 是什么 |
|---|---|
| `blacklist.json` | 手动维护的屏蔽名单(道具/关键词) |
| `solid_items.json` | `SolidItemFilter.py` 自动生成的,别手动改 |
| `item_stats.json` | `ComputeItemStats.py` 自动生成的,别手动改 |
| `PLEASE READ.txt` | 说明为什么贴图没放进仓库 |

### 关于贴图素材的说明

仓库里没放 `sprites/` 文件夹。Growtopia的道具贴图是游戏官方的版权素材,这个项目没有权利帮你分发这些图,所以需要你自己在本地准备一份。详细说明在 `output/sprites_note.txt` 里。
